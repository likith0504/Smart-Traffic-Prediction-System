# Smart Traffic Prediction and Congestion Management System

## Overview
A machine learning project that predicts traffic volume using weather and temporal features and provides intelligent congestion management recommendations.

## Features
- Traffic Volume Prediction
- Congestion Classification
- Anomaly Detection
- Feature Importance Analysis
- Adaptive Traffic Signal Recommendations

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Gradient Boosting Regressor

## Results
The model predicts traffic volume and classifies congestion levels to support smart traffic management systems.
